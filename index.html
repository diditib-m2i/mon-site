Suivre les modifications avec GIT

  1. modifier le fichier
  2. Vérifier les modifications avec git status
  3. Ajouter les changements avec git add
  4. Faire un commit avec un message clair git commit -m "message"
  5. Pousser les changements vers GitHub avec git push
